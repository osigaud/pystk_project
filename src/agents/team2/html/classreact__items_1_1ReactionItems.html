Detailed Description

Oriente le kart vers les bons items et l'éloigne des mauvais.

Priorité : esquiver un mauvais item proche l'emporte toujours sur aller chercher un bon item.

Bons items : BONUS_BOX(0), NITRO_BIG(2), NITRO_SMALL(3), EASTER_EGG(6). Mauvais items : tout le reste (banane, bubblegum...).

Retourne un ajustement de steering selon les items devant le kart.

Parcourt tous les items visibles (Z > 0, distance < 25). Pour les bons items : calcule l'angle vers le plus proche et ajuste le steering. Pour les mauvais items à moins de 15 unités : applique un angle d'esquive fixe.

Parameters
obsDictionnaire d'observation retourné par l'environnement. Utilise les clés "items_position" et "items_type".
Returns
float : ajustement de steering entre -1.0 et 1.0. 0.0 si aucun item pertinent n'est détecté.
