Detailed Description

Décide d'utiliser l'item offensif selon la position des adversaires.

Un tir est déclenché si un adversaire est à moins de 40 unités devant le kart et dans un cône de 15 degrés autour de l'axe longitudinal (z).

Vérifie si un adversaire est dans la zone de tir.

Parcourt tous les karts adversaires visibles dans l'observation, et retourne True dès qu'un adversaire remplit les deux conditions :

  • être devant le kart (z > 0)
  • être à moins de 40 unités et dans un cône de 15 degrés
Parameters
obsDictionnaire d'observation retourné par l'environnement. Doit contenir la clé "karts_position" (positions des autres karts dans le référentiel du kart courant).
Returns
bool : True si au moins un adversaire est dans la zone de tir, False sinon.
Note
Les karts derrière le kart (z <= 0) sont ignorés.
