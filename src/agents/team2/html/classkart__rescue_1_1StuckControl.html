Detailed Description

Surveille la vitesse du kart et déclenche un recul si nécessaire.

Si le kart reste sous le seuil de vitesse pendant trop de steps consécutifs, une séquence de marche arrière est activée pour le dégager. Doit être appelé en priorité absolue dans choose_action().
