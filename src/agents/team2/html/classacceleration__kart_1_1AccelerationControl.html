Detailed Description

Choisit le niveau d'accélération en fonction du virage détecté.

Appelle AnticipeKart.detectVirage() pour mesurer la courbure courante, puis retourne une accélération réduite selon le type de virage :

  • Virage très serré (> cfg.virages.drift) : 0.80
  • Virage serré (cfg.virages.serrer.i1 à i2) : 0.85
  • Virage moyen (cfg.virages.moyen.i1 à i2) : 0.95
  • Ligne droite (courbure faible) : 1.00
