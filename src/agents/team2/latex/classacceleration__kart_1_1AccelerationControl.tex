\doxysubsection{Detailed Description}
Choisit le niveau d\textquotesingle{}accélération en fonction du virage détecté. 

Appelle Anticipe\+Kart.\+detect\+Virage() pour mesurer la courbure courante, puis retourne une accélération réduite selon le type de virage \+:
\begin{DoxyItemize}
\item Virage très serré (\texorpdfstring{$>$}{>} cfg.\+virages.\+drift) \+: 0.\+80
\item Virage serré (cfg.\+virages.\+serrer.\+i1 à i2) \+: 0.\+85
\item Virage moyen (cfg.\+virages.\+moyen.\+i1 à i2) \+: 0.\+95
\item Ligne droite (courbure faible) \+: 1.\+00
\end{DoxyItemize}
