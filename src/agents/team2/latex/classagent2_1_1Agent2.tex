\doxysubsection{Detailed Description}
Agent de pilotage heuristique pour Super\+Tux\+Kart. 

Agent2 orchestre plusieurs sous-\/modules spécialisés pour piloter le kart \+:
\begin{DoxyItemize}
\item Steering\+Piste \+: maintien au centre de la piste.
\item Reaction\+Items \+: gestion des items (collecte / évitement).
\item Attack\+Rivals \+: utilisation des items offensifs contre les adversaires.
\item Stuck\+Control \+: détection et sortie des situations de blocage.
\item Acceleration\+Control \+: adaptation de l\textquotesingle{}accélération selon la courbure.
\end{DoxyItemize}

Calcule l\textquotesingle{}action complète à effectuer pour le step courant. 

Fusionne les décisions de tous les sous-\/modules dans l\textquotesingle{}ordre de priorité \+:
\begin{DoxyEnumerate}
\item Sortie de blocage (Stuck\+Control) — priorité absolue.
\item Steering vers le nœud cible (path\+\_\+lookahead nœuds devant).
\item Activation du rescue si le kart est hors piste.
\item Nitro si énergie disponible et trajectoire droite.
\item Correction de centrage (Steering\+Piste).
\item Accélération adaptée au virage (Acceleration\+Control).
\item Réaction aux items (Reaction\+Items).
\item Attaque des adversaires si item en main (Attack\+Rivals).
\end{DoxyEnumerate}
