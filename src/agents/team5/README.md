# Team 5 - Documentation Technique

### Utilisation du script generate_doc.sh

Pour generer ou mettre a jour la documentation, executez les commandes suivantes dans un terminal Linux (WSL/Ubuntu) :

- cd src/agents/team5 \
- chmod +x generate_doc.sh \
- ./generate_doc.sh

### Consultation

Une fois le script execute, ouvrez le fichier suivant dans un navigateur :
src/agents/team5/team5_docs/HTML/team5.html\
Note : Les fichiers HTML generes sont ignores par Git et ne doivent pas etre pushed sur le depot.
